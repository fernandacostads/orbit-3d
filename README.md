# 3D CSS Solar System

Solar System data visualisation done in HTML/CSS and a bit of TS.

## Modern development

Install dependencies:

```bash
npm install
```

Run the local dev server:

```bash
npm run dev
```

This project now uses TypeScript with Vite.

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
